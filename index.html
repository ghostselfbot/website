What is Ghost?
Ghost is a powerful open-source Discord selfbot that gives you better control over your Discord experience, lets you troll friends, and moderate servers from your own account.
Is Ghost free and open source?
Ghost is free and open source, and will stay open source for the foreseeable future. You can access the source on GitHub.
Which operating systems does Ghost support?
Ghost officially supports macOS and Windows. Some Linux distributions will work but require building from source. Ghost is built on macOS, so platform-specific issues may occur.
How do I install Ghost?
Use the download links above. To run Ghost from source, refer to the GitHub wiki.
Does Ghost require Python?
Ghost requires Python 3.10 through 3.13 when running from source.
Where can I find the latest version and release notes?
The latest version and release notes are available on GitHub Releases.
How often is Ghost updated?
There is no fixed update schedule for Ghost. Updates usually arrive a few months apart.
Why are updates not more frequent?
Ghost is a solo passion project maintained by Benny. Time is limited, but fresh updates and changes still come.
Is using Ghost against Discord's Terms of Service?
Using Ghost and other selfbots is against Discord's Terms of Service. Use it carefully; the project is not responsible for account termination caused by its use.
Can using Ghost restrict or terminate my account?
Because selfbots are against Discord's Terms of Service, there is a chance your account could be restricted or terminated. Understand and accept that risk before using Ghost.
Is Ghost affiliated with Discord?
No. Ghost is not affiliated with Discord.
Where can I get help or report a bug?
Get help or report bugs by creating a new GitHub issue.
How can I suggest a feature?
Create a GitHub issue or join the Discord server.
